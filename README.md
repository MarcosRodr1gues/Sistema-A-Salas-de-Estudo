# Sistema-A-Salas-de-Estudo
Uma aplicação desktop simples para gerenciar e reservar salas de estudo ou outros recursos (como projetores, calculadora etc) na faculdade. O problema principal é a dificuldade que os alunos enfrentam para encontrar espaços disponíveis e a falta de um sistema centralizado, o que gera conflitos de horário e uso ineficiente dos espaços.
